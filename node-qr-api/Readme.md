# QR Code API (Free)

### Author : Hariprasath.B

### Heroku App : https://node-qr-api.herokuapp.com

#### Get QR:
    - /qr/<QR-Content>

#### Get QR with Specific Size
    - /qr/<size-in-px>/<QR-Content>

#### Installation
    ```
    cd nqr/
    npm init
    touch app.js
    sudo npm i --save express
    sudo npm install --save qrcode
    ```
